## v1.0 デザイン仕様 (`d4f0e546`)

### レイアウト構成

```
┌─────────────────────────────────────┐
│  ← (戻る)           ⚙️ (設定/管理)  │ ← ヘッダー上のボタン
│                                     │
│      ヘッダー画像（グラデーション）     │ ← 150px高
│                                     │
├──────────────┬──────────────────────┤
│              │                      │
│    [Avatar]  │ ← 100px、白枠4px、中央配置
│              │   ヘッダーに50px重なる
└──────────────┴──────────────────────┘
          
          表示名（中央揃え）
        自己紹介テキスト（任意）

┌───────┬───────┬───────┐
│ 投稿  │ 称賛  │  徳   │ ← ラベル（上）
│  12   │  45   │  78   │ ← 数値（下、太字22px）
└───────┴───────┴───────┘

┌─────────────────────┬────┐
│     フォロー         │ ✉️  │ ← 紫グラデーション
└─────────────────────┴────┘
```

### コンポーネント詳細

#### ヘッダーセクション
- `Stack`構造でアバターをヘッダーに重ねる
- 戻るボタン（他ユーザー閲覧時のみ）
- 設定ボタン（自分のプロフィールのみ）
- 管理者メニューボタン（管理者のみ）

#### 名前・自己紹介セクション
- アバター下部から60pxのパディング
- 名前: `headlineSmall` + bold
- 自己紹介: `bodyMedium` + `textSecondary`

#### 統計セクション
- 3カラム均等配置（投稿・称賛・徳）
- ラベル上、数値下の構成
- パディング: 左右32px、上20px

#### フォローボタンセクション（他ユーザーのみ）
- 紫グラデーションボタン（横幅広め）
- メッセージアイコンボタン（右側、12px間隔）
- パディング: 左右24px、上20px
