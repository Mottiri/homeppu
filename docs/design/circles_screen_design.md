# サークル画面設計書

## 1. 概要
サークル一覧を表示し、検索・フィルタリング機能を提供する画面。

## 2. 画面構成 (UI Layout)

### 2.1 ヘッダー
*   **タイトル**: 「サークル」を中央に表示（シンプル）

### 2.2 検索バー
*   サークル名での検索が可能
*   白背景のカード型デザイン

### 2.3 タブセレクター
*   **みんなの**: 全公開サークル
*   **参加中**: 自分が参加しているサークルのみ

### 2.4 カテゴリチップ
*   水平スクロール対応
*   カテゴリでフィルタリング

### 2.5 サークルリスト
*   プル更新対応
*   無限スクロール対応（15件ずつ読み込み）
*   `_CircleCard`ウィジェットで表示

## 3. 機能仕様 (Functionality)

### 3.1 背景グラデーション (2026-01-06 追加)
*   ユーザーの`headerPrimaryColor`と`headerSecondaryColor`から動的にグラデーション生成
*   ホーム画面と同様のパステルカラー

### 3.2 スクロールトップ機能 (2026-01-06 追加)
*   ボトムナビのサークルボタンタップ時、既にサークル画面の場合はスクロールトップ
*   `circleScrollToTopProvider`でトリガー
