## 概要

**コアコンセプト**: 投稿 × タスク = 頑張りの可視化

---

## 1. マネタイズ設計

### 1.1 課金アイテム

| カテゴリ | 無料 | 課金限定 | 徳ポイント交換 |
|---------|------|----------|---------------|
| 投稿カード装飾 | 基本装飾 | 限定フレーム/エフェクト | レア装飾 |
| リアクションスタンプ | 基本絵文字 | 派手なアニメスタンプ | キラキラスタンプ |
| 名前カード | 基本デザイン | 限定デザイン | レアデザイン |
| サークルカード装飾 | 基本 | 限定 | - |
| アバター（将来） | 基本パーツ | 限定パーツ | レアパーツ |

### 1.2 設計ポイント

- **徳ポイント交換**：課金しなくても頑張れば手に入る（公平感）
- **課金限定品**：収益確保
- **カード装飾は自動で派手にならない**：自分で選んで装着

## 3. 週間チャレンジ機能

### 3.1 概要

AIがユーザーの投稿/サークル傾向からチャレンジを提案。
**証跡をAIに提出し、合格で報酬獲得。**

### 3.2 UID認証方式

**不正防止**：証跡写真にUIDを紙に書いて一緒に写す

```
チャレンジ達成時:
1. ユーザーがUIDを紙にメモ
2. 証跡写真にUIDメモを一緒に写す
3. AIが判定:
   ✅ UIDが写っている（自分のUID）
   ✅ チャレンジ内容と一致
   → 合格
```

### 3.3 AI判定フロー

1. 画像を受信
2. UID確認（なし → 不合格）
3. UID照合（不一致 → 不合格）
4. チャレンジ内容確認（関係ない画像 → 不合格）
5. 合格 → 報酬付与

### 3.4 チャレンジ例

| チャレンジ | 証跡 |
|-----------|------|
| 今週3回運動する | 運動中/後の写真 + UID |
| 本を10ページ読む | 本の写真 + ページ番号 + UID |
| 料理を1品作る | 完成した料理の写真 + UID |
| 早起き（6時前）を3回 | 朝の投稿（時間で検証） |

## 4. 集中ルーム

### 4.1 コア価値

```
❌ 何分集中したかの記録（抜け穴がある）
✅ 「一緒に頑張ってる仲間がいる」という安心感
```

### 4.2 機能

| 機能 | 優先度 | 内容 |
|------|--------|------|
| 仲間の存在表示 | **P1** | 「〇人が集中中」と表示 |
| 集中終了投稿 | P2 | 「〇〇分集中した！」自動投稿（任意） |
| タイマー | P3 | ポモドーロタイマー（補助的） |
| BGM | ❌ | YouTubeで代用可能 |

### 4.3 不正防止について

**集中ルームには報酬を紐づけない**。
単なる「居場所」として機能させるため、不正防止は不要。

## 5. AI関連の設計

### 5.1 AIコスト抑制

| 機能 | AI対応 | 理由 |
|------|--------|------|
| 投稿へのコメント | ✅（1回のみ） | 肯定の担保 |
| コメントへの返信 | ❌ | 会話が続くとコスト増+バレリスク |
| サークル内チャット | ❌ | 人間同士のみ |
| 音声ルーム | ❌ | 人間同士のみ |
| チャレンジ審査 | ✅（画像判定） | 1回の判定のみ |

### 5.2 AIバレ防止

- AIは1回だけ反応する
- 長い会話は発生しない設計
- サークル内は人間同士のみ

## 8. 開発・運用コスト

### 8.1 機能別コスト比較

| 機能 | 開発工数 | 運用コスト（MAU10,000時） | 備考 |
|------|---------|-------------------------|------|
| **サークル内チャット** | 3-5日 | **$5/月** | Firestore利用、低コスト |
| **音声ルーム** | 1-2週間 | **$300/月** | Agora等SDK利用、従量課金 |
| 集中ルーム | 2-3日 | $0 | 存在表示のみ |
| チェックイン機能 | 2-3日 | $0 | Firestore利用 |
| カード装飾 | 1週間 | $0 | 静的アセット |

### 8.2 チャット vs 音声 詳細比較

| 項目 | チャット | 音声ルーム |
|------|---------|-----------|
| 開発工数 | 3-5日 | 1-2週間 |
| 技術難易度 | **低** | 中〜高 |
| 追加ライブラリ | 不要（Firestore） | 必要（Agora等） |
| MAU 1,000時 | $0.50/月 | $30/月 |
| MAU 10,000時 | $5/月 | $300/月 |
| MAU 100,000時 | $50/月 | $3,000/月 |
| コスト比 | **1** | **60倍** |

### 8.3 音声ルーム料金詳細（Agora使用時）

- 無料枠: 1万分/月
- 超過分: $0.99/1000分
- 計算式: ユーザー数 × 1日利用分数 × 30日 × 単価

### 8.4 結論

- **チャット優先**：低コストで滞在時間増加
- **音声は後回し**：収益安定後に検討
